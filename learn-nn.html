Neural Networks

Computers use multilayer networks to simulate a brain. Each "Node" has a list of outputs, which is matched with a list of weights. These work together to output a number to the next list of Nodes.

See Demo »

Taking Inputs

When a Node takes an input, it adds it to the sum it is already tracking. The function below allows other Nodes to give the specific Node an input.

It is a very simple function.

input(i) {
  this.sum += i;
}

The Sigmoid

The Sigmoid function is vital to Neural Networks. It allows the nodes to have values much larger that 1, yet only output a maximum of 1. The function is usually defined as 1/(1+2^(-x)).

However, in this project, the function has been modified to return a value between -1 and 1. 

function sigmoid(n) {
  return 2*(1/(1+pow(5, 0-n)))-1;
}

Giving Outputs

When a Node has received all of it's inputs, it then outputs to the Nodes it is connected to.

Like the input function, it is relatively simple. 

output() {
  let out = sigmoid(this.sum);
  for (let i = 0; i<this.outs.length; i++) {
    this.outs[i].input(out*this.weights[i]);
  }
}
