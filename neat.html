    Sensing Surroundings

    In order for the algorithm to work, it must be able to know what is going on around it.

    It then gives this information to the brain.

    In this simulation, the network takes an input of a list, and then sends the values to their respective Nodes.

    this.brain.clear();
    let data = [this.hgt,cHeight,nDist,nHeight,1]
    this.brain.update(data);
    if (this.brain.outputs[0].sum > 0) {
      this.jump();
    }

    Tweaking Connections

    Like in the Genetic Algorithm Demonstration, the bots need to mutate in order to learn. In this game, the bots mutate by having their connections altered slightly

    mutate() {
      for (let i=0;i<this.inputs.length;i++) {
       for(let w=0;w<this.inputs[i].weights;w++){
          if (random(1)<0.05) {
            let temp = (random(2)-1);
            this.inputs[i].weights[w] = temp;
          }
        }
      }
      for (let i=0;i<this.calcs.length;i++) {
       for(let w=0;w<this.calcs[i].weights;w++){
          if (random(1)<0.05) {
            let temp = (random(2)-1);
            this.inputs[i].weights[w] = temp;
          }
        }
      }
    }
